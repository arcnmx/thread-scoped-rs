# thread-scoped

[![travis-badge][]][travis] [![release-badge][]][cargo] [![docs-badge][]][docs] [![license-badge][]][license]

A `std::thread::spawn()` that can access its current scope.
Stable fork of the deprecated `std::thread::scoped()`
